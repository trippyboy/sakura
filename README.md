# 実行方法

引数を指定した場合は、その値が使用率閾値として利用される

```
$ sh mailbox_size_list.sh 99
user2 100.00% (1.95GB/1.95GB)
user7 99.99% (699.99MB/700.00MB)
user1 99.99% (499.99MB/500.00MB)
$
$ sh mailbox_size_list.sh 80
user4 83.74% (167.49MB/200.00MB)
user9 81.94% (163.89MB/200.00MB)
user8 97.51% (195.02MB/200.00MB)
user2 100.00% (1.95GB/1.95GB)
user7 99.99% (699.99MB/700.00MB)
user1 99.99% (499.99MB/500.00MB)
```

引数を指定しない場合は、閾値が 0 として処理される

```
$ sh mailbox_size_list.sh
user1 99.99% (499.99MB/500.00MB)
user2 100.00% (1.95GB/1.95GB)
user3 50.00% (100.00MB/200.00MB)
user4 83.74% (167.49MB/200.00MB)
user5 50.00% (100.00MB/200.00MB)
user6 50.00% (100.00MB/200.00MB)
user7 99.99% (699.99MB/700.00MB)
user8 97.51% (195.02MB/200.00MB)
user9 81.94% (163.89MB/200.00MB)
```
